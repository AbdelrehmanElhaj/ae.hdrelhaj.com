## Tech

- Pure HTML + CSS + Vanilla JS — no framework, no build step
- Fonts: Space Grotesk (display) + Inter (body) via Google Fonts
- Images embedded as base64 WebP — fully self-contained single file
- Deployed via Cloudflare Workers (static assets)

## Deployment

```bash
git add .
git commit -m "Update"
git push origin main
```

Cloudflare auto-deploys on every push to `main`.
